Name

email — An e-mail address

Synopsis

Content Model

(#PCDATA)

Attributes

NameTypeDefault
targetsCDATANone

Description

The email element specifies a single e-mail address.

Parents

email may be contained in these elements: contact

See Also

fax, instantMessage, pager, url.

Examples
